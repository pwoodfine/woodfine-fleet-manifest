# SPOKE CONFIGURATIONS LEDGER

**EXECUTING ENTITY:** Woodfine Management Corp
**PROCUREMENT VENDOR:** PointSav Digital Systems

This directory houses the cryptographic payloads and operational manifests required to connect remote digital assets to `fleet-infrastructure-leased` (The Sovereign Hub).

## DIRECTORY TAXONOMY

### 1. Cryptographic Payloads (The Keys)
* `*.conf` files: Native WireGuard configuration files for macOS/Linux endpoints.
* `*.png` files: Optical QR code payloads for rapid iOS/Android mobile deployment.

### 2. General Operational Manifests (Support Desk)
These guides document the generalized deployment parameters across hardware classes. They are utilized by Woodfine support personnel to understand the underlying infrastructure.
* `GUIDE-macOS-endpoints.md`: The generic Apple desktop/laptop provisioning guide.
* `GUIDE-iOS-Android-endpoints.md`: The mobile optical provisioning guide.

### 3. Individual Operator Manifests (End-Users)
These guides are stripped down, zero-friction instruction sets tailored to specific geographic and hardware realities for individual Woodfine operators.
* `GUIDE-peter-macbook.md`: Targeted remote deployment instructions for Peter's MacBook Air (Mexico).
